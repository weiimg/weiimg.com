台灣人設計的字型也要取諧音梗

2024.09.30 · WEIIMG

你好👋這裡是 img letter，會分享一些我最近發現的事。

目前想把這個電子報定調成：「你大概不知道但對你說不定有幫助」的版面，可能會有些雜亂，如果看完以下有些心得的話，也歡迎直接回覆這封信，我會回ㄉ✒️字型在這幾年好像是一個滿泛用的主題，剛好最近看到不錯的內容，以下跟你們分享今年推出的兩套中文字型：

🦆扁桃體

好前陣子看到一款中國的開源字型「得意黑 Smilely Sans」，是平常少見「有個性」、「易讀性高」的免費黑體。

這種黑體除了 Dela Gothic One 外真的很少，但很可惜的是得意黑不支援繁體中文。

就在 9 月初，字嗨有位成員發布了用得意黑擴充的繁體字型，命名為「扁桃體」，真的是繁體中文字型社群一大福音。

不過目前使用有發現一些問題，例如筆畫、字距、字寬、中宮（註）等在某些字上會出現很明顯的錯誤，有些會卡在一起。

「去」、「亂擺」、「型」的比例、轉角等細節比較看得出來問題在海報、標準字等平面設計使用上不太影響，畢竟本來就該做細節調整。不過影片要上字卡的話滿困擾的，剪輯軟體的字距沒辦法單獨做調整。

總之是又多了一款好用字體，只可惜沒放到 Github 上好好整理呈現。

像原字體「得意黑」以及之前由 Dela Gothic One 補充而成的「饅頭黑體」就有在 Github 上整理，剛好在撰寫本文的時候也發現，作者饅頭本人上星期做了小更新。

🌵粗線體

由 壹加壹一粒 與 justfont 推出的「粒線體」續作 —「粗線體」，同樣是手寫體，不過用途比較不一樣，這邊以影片應用舉例。

粒線體

粗線體

字體特徵筆畫較細、筆畫末端延伸較長筆畫較粗、筆畫末端無特別延伸適用YouTube 封面、影片字卡封面、短影音字幕、字卡不適用字幕（字小不清楚）

長影音字幕（看久會視覺疲乏）

最近很常看到短影音字幕直接用粗線體來上，原本以為會不好閱讀，尤其是在剪接節奏很快的影片上面，不過我看到兩個優點：

  1. 夠粗

  2. 夠 Lifestyle

夠粗

夠 Lifestyle

這套字在足夠的娛樂性、美術性下，也保持一定的明視度／辨識度。

粗線體推出的時機也很剛好，正好是短影音成長到了一定階段，有一定的市場願意讓創作者投資購買工具／資源，可以說是為了中文短影音打造的一款字型了。

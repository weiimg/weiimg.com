電腦容量不夠了，怎麼買硬碟

2024.10.06 · WEIIMG

你好👋這裡是 img letter，會分享一些我最近發現的事。

目前想把這個電子報定調成：「你大概不知道但對你說不定有幫助」的版面，這次分享的是硬體科普，如果有文章有錯誤或想詢問的，都歡迎直接回覆這封信，我會回ㄉ✒️目前筆電市場上不知道為什麼都只給 1TB SSD 的儲存空間，雖然比老筆電快很多，但為了效能連 2.5” 的 HDD 都不給了，也不知道給不給人擴充，導致很多人電腦用不到 1 年，C 槽就已經紅到發燙了。

如果你是以下這幾種人，請繼續看下去

  • 想帶出去：工作需求，需要一直帶著大量檔案外出

想帶出去：工作需求，需要一直帶著大量檔案外出如果你是對硬體完全毫無概念的人，只要知道「HDD」跟「SSD」這兩個東西是什麼就好，它們分別由硬碟的結構分為以下兩種：

HDD（傳統硬碟）

3.5" 吋的 SATA 接口 HDD

  • 優點：便宜、容量大。

  • 缺點：速度慢、有移動零件，較容易有物理損壞。

  • 適合：備份大量資料。

優點：便宜、容量大。

缺點：速度慢、有移動零件，較容易有物理損壞。

適合：備份大量資料。

SSD（固態硬碟）

M.2 接口的 SSD

  • 優點：速度快、啟動快。

  • 缺點：價格較高、容量相對較小。

  • 適合：遊戲、影片剪輯、快速讀寫。

優點：速度快、啟動快。

缺點：價格較高、容量相對較小。

適合：遊戲、影片剪輯、快速讀寫。

那麼接下來就介紹如何選擇適合你的硬碟：

（p.s. 此篇以「擴充額外儲存空間」說明，不更換原有硬體配置，實際使用還是可以問一下電腦店。）

🎮想要玩遊戲、剪影片

大量讀寫、快速讀寫需求 → 選擇 SSD（固態硬碟）

很多人遊戲會裝在 C 槽或 D 槽，其實我都不太推薦，因為：

  • C 槽是系統碟，空間佔用太多會導致整台電腦性能變差

C 槽是系統碟，空間佔用太多會導致整台電腦性能變差

  • D 槽一般是 HDD，讀寫相對較慢，遊戲載入速度也會慢

D 槽一般是 HDD，讀寫相對較慢，遊戲載入速度也會慢以主流遊戲來說，一款遊戲大約會需要 20～80 GB 的儲存空間，可以根據你玩的遊戲數量選擇硬碟容量，推薦可以買 512 GB／1 TB 的 SSD。

以剪輯影片來說，一個專案會有 50～200 GB 以上的素材都有可能，在剪輯一定要有足夠的讀寫速度，讓你工作效率更好，推薦可以買 2TB 以上的 SSD。

🗃️想要備份資料

重要資料、大量備份需求 → 選擇 HDD（傳統硬碟）

備份資料對於現代人來說是必須，但很常被忽略的事，等到要救資料的時候才開始頭痛。

如果你大部分的資料，平常不會做高頻率的存取，HDD 會是好選擇。

因為 HDD 是使用物理結構進行存取，要救資料是相對比較救得回來的，可能放個十年都不會壞。（但還是需要定期備份最安全）

🧳需要帶一堆檔案出門

這時候就需要考慮到攜帶性、大小以及抗震性了，而你有兩種選擇：

1. 行動硬碟

為了市場面向推出的行動硬碟，不會有什麼大問題，基本上主流大廠可以直接在預算內無腦選。

2. 硬碟＋外接盒

在選擇上有更高的彈性，最重要的是硬碟的讀寫數值會比較透明公開，相比行動硬碟可以選擇更好的讀寫性能。

另外外接盒購買主要注意 2 點：

  • 尺寸：搭配硬碟尺寸（2.5”、3.5”、M.2）

尺寸：搭配硬碟尺寸（2.5”、3.5”、M.2）

  • 材質：選擇散熱較好的鋁合金外殼

材質：選擇散熱較好的鋁合金外殼
